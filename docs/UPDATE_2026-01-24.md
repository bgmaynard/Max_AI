# MAX_AI Scanner Service - Update Summary
## January 24, 2026

### Overview
Major enhancements to the real-time stock scanner dashboard including Finviz integration, trading halts monitoring, and improved UI/UX with persistent data handling.

---

## New Features

### 1. Finviz Small Cap Gainers Integration
- **New Panel Type**: Added "Finviz Gainers" as a data source option for dashboard panels
- **Full Coverage**: Returns ALL top gainers (163+ stocks) including penny stocks
- **Smart Sorting**: Results sorted by % change descending (biggest movers first)
- **Auto Universe Sync**: New symbols automatically added to scanner universe for real-time tracking
- **Endpoints**:
  - `GET /finviz/gainers` - Fetch all top gainers with optional price/limit filters

### 2. Trading Halts Scanner
- **New Module**: `scanner_service/ingest/halt_tracker.py`
- **Real-Time Monitoring**: Fetches from NASDAQ trade halts RSS feed every 15 seconds
- **Halt Information Tracked**:
  - Symbol
  - Halt time
  - Halt price
  - Halt reason
  - Resume time
  - Resume price
  - Duration (auto-calculated)
  - Exchange
- **Manual Entry Support**: Add halts manually for testing or tracking
- **Endpoints**:
  - `GET /halts` - All current and recent halts
  - `GET /halts/active` - Only currently halted stocks
  - `GET /halts/resumed` - Recently resumed halts
  - `POST /halts/add` - Manually add a halt
  - `POST /halts/{symbol}/resume` - Mark a halt as resumed

### 3. Resizable & Moveable Panels
- **Drag to Move**: Click and drag panel headers to reposition
- **Resize Handles**: Corner handles for resizing panels
- **Absolute Positioning**: Panels now use absolute positioning for free-form layout
- **Persistent Layout**: Panel positions saved with panel configuration

### 4. Sticky Stock List
- **Persistent Data**: Stocks remain in the list across data refreshes
- **No More Blinking**: In-place DOM updates prevent visual flickering
- **Stale Indicator**: Stocks not in latest fetch shown with reduced opacity
- **Clear List Button**: Yellow trash icon to manually clear and reset the list
- **Sorted Updates**: New stocks inserted in correct position by sort criteria

## Bug Fixes

### NaN JSON Serialization Error
- **Issue**: `ValueError: Out of range float values are not JSON compliant: nan`
- **Fix**: Added `safe_float()` and `safe_int()` helper functions to handle NaN/Infinity values

### Finviz Limited Results
- **Issue**: Only 6 stocks showing when 163+ available
- **Fix**: Changed from Ownership to Overview screener, increased limit to 500

### Missing Penny Stocks
- **Issue**: Cheapest stock showing was $29 instead of penny stocks like DRCT ($4.11)
- **Fix**: Increased max_price to 1000 and limit to 500 in frontend parameters

### Filter Data Loss
- **Issue**: Applying filters caused data to disappear
- **Fix**: Changed to fetch broad data from API, apply filters client-side only

### Alphabetical Sorting
- **Issue**: Results were sorted alphabetically instead of by % change
- **Fix**: Added `order='Change', ascend=False` to screener and explicit sort

### Safe Number Handling
```python
def safe_float(val, default=0.0):
    if val is None: return default
    f = float(val)
    return default if math.isnan(f) or math.isinf(f) else f
```
